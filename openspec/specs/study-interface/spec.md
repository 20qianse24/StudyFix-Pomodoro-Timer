## ADDED Requirements

### Requirement: Minimalist Layout
The system SHALL feature a centered layout with a modern, focused 'study vibe'.

#### Scenario: Visual presentation
- **WHEN** the user views the application
- **THEN** all core elements (Timer, Presets, Controls) SHALL be centered on the screen with consistent spacing.

### Requirement: Active Preset Highlighting
The system SHALL visually distinguish the currently selected preset button.

#### Scenario: Highlighting 30m preset
- **WHEN** the 30-minute preset is active
- **THEN** the 30-minute button SHALL have a distinct visual style (e.g., background color change or border) compared to the other preset buttons.

### Requirement: Soft Color Palette
The interface SHALL use a soft color palette to minimize distractions and promote focus.

#### Scenario: Theme application
- **WHEN** the page is loaded
- **THEN** the CSS SHALL apply a color scheme consisting of muted or pastel colors.
